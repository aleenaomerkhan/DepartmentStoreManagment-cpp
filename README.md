# DepartmentStoreManagment
Programming Fundamentals Project
A console-based application developed in C++ using Programming Fundamentals concepts. The system manages product records through file handling and provides features to add, view, search, update, and delete products. It also includes a billing module to calculate total amount for customer purchases using a menu-driven interface.
